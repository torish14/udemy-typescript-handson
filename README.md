# ハンズオンで学ぶ TypeScript

## 学んだことや気づきを得たこと

- `git push origin HEAD` で今いるブランチの origin をリモートに push する
- `git checkout -` で前回いたブランチに移動できる
- `npm info` で npm コマンドの情報を詳しく知れる
- `ts-node-dev --respawn` で TypeScript ファイルが編集されたとき、自動で Javascript にトランスパイルできる
